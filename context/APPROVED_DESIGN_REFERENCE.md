# Kiwi — Approved Design Reference

The approved direction is a warm editorial personal planner inspired by the supplied reference boards and the final Kiwi concepts.

## Non-negotiable visual decisions

- Warm paper/ivory background, never a plain Material white screen.
- Expressive editorial headings paired with highly readable sans-serif content.
- Layered organic color fields, subtle paper grain, and restrained botanical line art.
- Rich coordinated palette: coral/rose, peach, butter, sage/forest, powder blue, periwinkle, and lavender.
- Rounded cards with deliberate variation; do not render every section as an identical white rectangle.
- Floating bottom dock navigation with a strong selected state.
- One memorable focal element per screen.
- Calm, feminine, nature-inspired, mature, and useful—not childish or noisy.

## Approved screen families

1. Welcome: emotional hero statement, botanical/organic composition, one clear start action.
2. Today: greeting, date context, “make today feel good” hero, progress, prioritized tasks, self-care prompt, floating dock.
3. Calendar: horizontal date selector, category-coded timeline, clear current-time marker, add action.
4. Add task: focused form, soft grouped fields, priority chips, clear create action.
5. Wellness: recorded facts, gentle historical summaries, no predictions.
6. Diary: private writing-led surface with photo/mood support.
7. Ask Kiwi: bounded command interface with suggestions and confirmation.

## Reference usage

Reference images are visual targets for hierarchy, color rhythm, density, and emotional feel—not assets to copy literally. Each implementation task must compare the rendered screen against the relevant reference before completion.

## Visual acceptance test

Reject a screen if it looks like a generic Material template, uses a flat white background, has no focal point, uses all accents equally, or loses readability on a tablet.
