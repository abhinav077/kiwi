# Kiwi Visual Reference Folder

This folder contains the generated Kiwi concepts and the original visual references supplied during design exploration.

Use these images to compare hierarchy, color rhythm, rounded surfaces, timeline density, and emotional feel. They are not production assets and must not be copied literally. Production assets are in `../assets/vectors/`.

When a screen is implemented, record the reference filename used in the task completion report and update `PROGRESS_TRACKER.md` if a visual decision changes.
